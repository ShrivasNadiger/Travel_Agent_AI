# Tourism Frontend Only

A React + Vite frontend for the Tourism Multi-Agent System.

## Setup

1.  Install dependencies:
    ```bash
    npm install
    ```

2.  Run development server:
    ```bash
    npm run dev
    ```

3.  Open in browser:
    http://localhost:5173

## Note
This frontend is configured to connect to the backend at `http://localhost:3000/api/v1/tourism`. Ensure the backend server is running.
